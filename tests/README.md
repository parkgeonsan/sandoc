# 테스트 데이터

## 용도

이 폴더에 실제 HWP/HWPX 파일을 넣어 sandoc 시스템을 테스트합니다.

---

## 필요한 테스트 파일

테스트를 위해 다음 파일들을 이 폴더에 넣어주세요:

| 파일 | 설명 | 용도 |
|------|------|------|
| `sample-공고문.hwp` | 실제 정부 지원사업 공고문 | 서류 분류 & 공고 분석 테스트 |
| `sample-양식.hwp` | 사업계획서 양식 (빈 양식) | 양식 구조 추출 & HWPX 출력 테스트 |
| `sample-사업자등록증.jpg` | 사업자등록증 스캔/사진 | OCR & 프로필 자동 추출 테스트 |

---

## 주의사항

- **실제 회사 정보가 포함된 파일은 `.gitignore`로 관리됩니다**
- 이 폴더의 파일은 Git에 커밋하지 마세요 (민감 정보 포함 가능)
- 테스트용 더미 데이터를 별도로 만들어 사용하는 것을 권장합니다

---

## 테스트 시나리오

1. **서류 분류 테스트**: `sample-공고문.hwp`와 `sample-양식.hwp`를 넣고 "분석해줘" 실행
2. **양식 파싱 테스트**: `sample-양식.hwp`의 섹션 구조가 정확히 추출되는지 확인
3. **HWPX 출력 테스트**: 양식에 텍스트 삽입 후 서식이 보존되는지 확인
4. **OCR 테스트**: `sample-사업자등록증.jpg`에서 회사 정보가 추출되는지 확인
