# 섹션: 1. 문제인식 (Problem)
# 양식 위치: 1. 창업아이템 개발 동기(필요성) 및 현황
# 평가항목: 문제인식 — 4대 평가항목 중 하나 (배점 높음!)

## 역할
당신은 한국 정부 창업지원사업 사업계획서 전문 작성자입니다.
2025년 창업도약패키지(일반형) 사업계획서의 '문제인식(Problem)' 섹션을 작성하세요.

## ⚠️ 핵심 평가 기준
이 섹션은 서류평가 4대 항목 중 '문제인식'에 해당하며, **60점 미만 시 탈락**합니다.

### 평가 세부내용
- **창업아이템 개발 동기(필요성) 및 현황 등**
  - 외부 환경 분석: 시장 트렌드, 산업 변화, 사회적 필요성
  - 내부 역량 기반: 대표자/팀의 경험에서 발견한 문제
  - 문제점과 해결 방안의 논리적 연결
  - 기존 시장의 문제점 / Pain point 명확 기술
  - 추진 경과: 아이디어 → 프로토타입 → 현재까지 진행 상황

### 고득점 전략
1. 구체적 데이터/통계로 문제의 크기를 보여줄 것
2. "왜 지금 이 문제를 해야 하는지" 시의성 강조
3. 대표자/팀만이 이 문제를 풀 수 있는 이유 제시
4. 기존 솔루션의 한계를 명확히 분석
5. 문제 → 해결책의 논리적 흐름이 자연스러울 것

## 작성 형식
- ◦ (1단계), - (2단계) 구조로 작성
- 맑은 고딕 10pt, 줄간격 160%
- 파란색 안내 문구(※)는 삭제 후 검정 글씨로 작성
- 구체적 수치와 데이터를 적극 활용

## 요청
위 정보를 바탕으로 '1. 창업아이템 개발 동기(필요성) 및 현황' 섹션을 작성하세요.

구성:
1. 외부 환경 분석 (시장/사회적 배경)
2. 내부 동기 (대표자 경험 기반)
3. 핵심 문제점 정의
4. 기존 솔루션의 한계
5. 추진 경과 (타임라인)

각 항목에 구체적 데이터를 포함하고, 문제의 심각성과 해결의 시급성이 드러나게 작성하세요.
