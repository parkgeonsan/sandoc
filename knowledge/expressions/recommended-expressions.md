# 사업계획서 권장 표현 사전

## 개요

사업계획서에서 자주 사용되는 표현을 상황별로 정리한다.
AI 생성 텍스트에서 흔히 나타나는 부자연스러운 표현을 교정하고,
심사위원에게 신뢰감을 주는 표현으로 대체하는 데 활용한다.

---

## 1. 필요성/당위성 서술

### 권장 표현

| 상황 | 권장 표현 | 비고 |
|------|----------|------|
| 문제 제기 | ~가 시급한 과제로 대두되고 있다 | |
| 시장 수요 | ~에 대한 수요가 지속적으로 증가하고 있다 | 수치 근거 필수 |
| 정책 부합 | 정부의 [정책명] 기조에 부합하는 사업이다 | |
| 기술 한계 | 기존 [기술]은 [한계점]의 구조적 한계를 가지고 있다 | |
| 해외 의존 | 핵심 [기술/소재]의 해외 의존도가 [X]%에 달하는 실정이다 | |
| 경쟁력 약화 | 국내 [산업]의 글로벌 경쟁력 확보가 필요한 시점이다 | |
| 사회 변화 | [사회 변화]로 인해 [분야]의 패러다임이 전환되고 있다 | |

### 피해야 할 표현 → 대체

| 피할 것 | 대체 표현 |
|---------|----------|
| ~의 필요성은 두말할 나위가 없다 | ~의 필요성이 확인된다 |
| 모두가 알다시피 | [출처]에 따르면 |
| 당연히 ~해야 한다 | ~이 요구된다 / ~이 시급하다 |
| ~은 자명하다 | ~으로 확인된다 / ~으로 분석된다 |

## 2. 기술 설명

### 권장 표현

| 상황 | 권장 표현 |
|------|----------|
| 기술 원리 | 본 기술은 [원리]에 기반하여 [효과]를 달성한다 |
| 성능 비교 | 기존 [기술] 대비 [지표]가 [X]% 향상/절감된다 |
| 독창성 | 본 기술의 핵심 차별점은 [차별점]에 있다 |
| 기술 수준 | 현재 TRL [X] 단계이며, 본 과제를 통해 TRL [Y]까지 달성한다 |
| 특허 기반 | 자체 보유 특허 [X]건을 기반으로 기술 독자성을 확보하였다 |
| 하이브리드 | [기술A]와 [기술B]를 융합한 하이브리드 방식을 적용한다 |
| 확장성 | 본 기술은 [분야A], [분야B] 등으로의 확장이 가능하다 |
| 검증 결과 | [환경] 조건에서 [테스트]를 수행한 결과, [결과]를 확인하였다 |

### 피해야 할 표현 → 대체

| 피할 것 | 대체 표현 |
|---------|----------|
| 세계 최초의 기술 | 국내 최초 수준의 / 선행 사례가 확인되지 않는 |
| 혁신적인 기술 (반복) | [수치]를 달성하는 차별화된 기술 |
| 매우 뛰어난 성능 | [지표] 기준 [수치]의 성능 |
| 완벽한 솔루션 | 검증된 솔루션 / 안정적인 솔루션 |
| 무한한 가능성 | [N]개 분야로의 확장이 가능하다 |
| 최첨단 기술 | 선도적인 기술 / 차세대 기술 |

## 3. 시장/사업화 서술

### 권장 표현

| 상황 | 권장 표현 |
|------|----------|
| 시장 규모 | [산업] 시장은 [연도] 기준 약 [X]조 원 규모이다 (출처: [기관]) |
| 성장률 | 연평균 [X]%의 성장률(CAGR)을 기록하고 있다 |
| 타겟 설정 | TAM [X]조 원, SAM [Y]억 원, SOM [Z]억 원으로 산출된다 |
| BM 설명 | [고객]에게 [제품/서비스]를 [채널]을 통해 제공하는 비즈니스 모델이다 |
| 매출 추정 | 보수적 시나리오 기준, 사업화 후 [X]년 내 매출 [Y]억 원을 목표로 한다 |
| 경쟁 분석 | 주요 경쟁사 [X]개 사의 제품을 비교 분석한 결과 |
| 고객 확보 | 초기 고객 [X]개 사를 확보하였으며(MOU/LOI 체결) |
| 수출 계획 | [대상국]을 중심으로 해외 시장 진출을 추진한다 |

### 피해야 할 표현 → 대체

| 피할 것 | 대체 표현 |
|---------|----------|
| 시장이 매우 크다 | [X]조 원 규모의 시장이다 |
| 폭발적 성장 | CAGR [X]%의 고성장 |
| 경쟁자가 없다 | 직접 경쟁사 [X]개, 대체재 포함 [Y]개가 존재한다 |
| 독보적인 위치 | 시장 점유율 [X]%를 확보하고 있다 |
| 블루오션 시장 | 경쟁 강도가 낮은 시장 / 신규 형성 중인 시장 |

## 6. 전환어/접속어

### 논리 전개용

| 용도 | 표현 |
|------|------|
| 추가 | 또한, 아울러, 나아가, 더불어 |
| 대조 | 반면, 그러나, 한편, 이에 반해 |
| 인과 | 이에 따라, 따라서, 이를 통해, 그 결과 |
| 강조 | 특히, 무엇보다, 주목할 점은, 핵심은 |
| 예시 | 예를 들어, 구체적으로, 일례로 |
| 요약 | 종합하면, 요약하면, 결론적으로 |
| 순서 | 첫째/둘째/셋째, 우선/다음으로/마지막으로 |

### 피해야 할 전환어

| 피할 것 | 이유 | 대체 |
|---------|------|------|
| 그리고 (문장 시작) | 구어체 | 또한 / 아울러 |
| ~뿐만 아니라 ~도 (반복) | 단조로움 | 나아가 / 더불어 |
| 물론 | 불필요한 양보 | (삭제하거나 "한편"으로 대체) |
| 사실은 | 이전 서술의 신뢰성 저해 | (삭제) |

## 7. AI 생성 텍스트 교정 패턴

AI(LLM)로 작성한 초안에서 흔히 나타나는 표현과 교정 방향을 정리한다.

| AI 특유 표현 | 교정 방향 | 예시 |
|-------------|----------|------|
| ~에 있어서 | ~에서 / ~에 대해 | "기술 개발에 있어서" → "기술 개발에서" |
| ~라 할 수 있다 | ~이다 (직접 서술) | "핵심이라 할 수 있다" → "핵심이다" |
| ~것은 자명하다 | ~것으로 확인된다 | |
| ~할 필요가 있다 | ~해야 한다 / ~이 필요하다 | |
| 다양한 (남발) | 구체적 수치/목록으로 대체 | "다양한 분야" → "[A], [B], [C] 등 [N]개 분야" |
| ~를 통해 ~를 통해 (중복) | 한 번만 사용 | |
| 이러한 관점에서 볼 때 | (삭제 또는 간결화) | |
| ~적인 (과도 사용) | 적절히 조절 | "혁신적인 기술적인 접근" → "혁신적 기술 접근" |
| 결론적으로 (문서 중간) | 요약하면 / 종합하면 | |
| ~기 위해서는 ~해야 한다 | ~하려면 ~해야 한다 | 간결화 |
