# 심사 배점 체계

## 개요

정부 지원사업 사업계획서 심사는 **정량 평가**와 **정성 평가**로 구성된다.
배점표를 정확히 분석하고, 고배점 항목에 전략적으로 집중하는 것이 핵심이다.
각 사업마다 심사 기준이 다르므로, 공고문에 명시된 배점표를 반드시 확인하고 이에 맞춰 전략을 수립해야 한다.

---

## 일반적인 심사 항목 구조

대부분의 정부 지원사업은 다음과 같은 대항목으로 구성된다:

### 공통 심사 항목

| 대항목 | 일반 배점 범위 | 핵심 포인트 |
|--------|-------------|------------|
| 기술성 | 20~35점 | 기술의 차별성, 개발 가능성, 독창성 |
| 사업성 | 15~30점 | 시장 규모, 매출 계획, 사업화 전략 |
| 대표자/팀 역량 | 10~25점 | 경력, 실적, 전문성, 팀 구성 |
| 사업 타당성 | 10~20점 | 예산 적정성, 추진 일정, 실현 가능성 |
| 정책 부합성 | 5~15점 | 정부 정책 방향과의 일치도 |
| 가점 | 0~10점 | 특허, 인증, 고용, 지역, 여성/장애인 등 |

### 배점 분석 원칙

1. **고배점 항목 우선** — 30점짜리 기술성에 40%의 노력 투자
2. **감점 방지** — 빈칸, 오탈자, 수치 불일치는 직접 감점
3. **가점 극대화** — 해당되는 가점 항목 빠짐없이 챙기기
4. **과락 기준 확인** — 특정 항목 60% 미만이면 탈락하는 경우 있음

## 주요 사업별 상세 심사 배점

### 1. 창업도약패키지 (중소벤처기업부)

창업 3~7년차 기업 대상, 최대 3억 원 지원.

| 심사 항목 | 배점 | 세부 평가 요소 |
|-----------|------|---------------|
| 대표자 역량 | 20점 | 창업 동기, 관련 분야 경험, 리더십, 사업 추진 의지 |
| 기술/아이디어 혁신성 | 25점 | 기술 독창성, 차별화 정도, 지식재산권 보유 현황, 기술적 실현 가능성 |
| 사업화 역량 및 성과 | 25점 | 매출 실적 및 성장률, 시장 분석, 비즈니스 모델 타당성, 목표 시장 명확성 |
| 성장 전략 | 20점 | 향후 성장 로드맵, 자금 운용 계획, 투자 유치 계획, 해외 진출 전략 |
| 정책 부합성 | 10점 | 일자리 창출, 사회적 가치, 지역 균형 발전 |

**창업도약패키지 전략 포인트:**
- "도약"을 키워드로 삼되, 기존 3~7년간의 성과를 근거로 "성장 잠재력"을 입증해야 한다
- 매출 실적이 있는 기업 대상이므로, 초기 성과 데이터가 핵심이다
- 자금 사용 계획의 구체성이 성장 전략 배점에 크게 영향을 미친다
- 발표 평가가 있으므로 사업계획서와 발표자료의 일관성이 중요하다

### 2. TIPS (민간투자주도형 기술창업지원)

엔젤투자자/VC 추천 기반, R&D 최대 5억 원 + 사업화 1억 원.

| 심사 항목 | 배점 | 세부 평가 요소 |
|-----------|------|---------------|
| 기술 혁신성 | 30점 | 원천 기술 보유 여부, 특허/논문, 기술적 차별성, 글로벌 경쟁력 |
| 대표자/팀 역량 | 25점 | 기술 개발 경력, 핵심 인력 구성, 관련 분야 전문성 |
| 시장성/사업성 | 25점 | 글로벌 시장 규모, BM 타당성, 수익 모델, 시장 진입 전략 |
| 투자 타당성 | 20점 | 투자 매력도, 성장 가능성, Exit 전략, 후속 투자 가능성 |

**TIPS 전략 포인트:**
- 민간 투자자(운영사)가 1차 선별하므로, 운영사 관심 분야를 파악해야 한다
- "글로벌 시장"을 타겟으로 한 기술 기업에 유리한 구조다
- 기술의 독창성과 IP 확보 현황이 매우 중요하다
- 투자 유치 경험이나 계획이 있으면 가점 요소로 작용한다
- R&D 계획의 구체성 (단계별 마일스톤, 기술 목표치)이 핵심이다

### 3. 예비창업패키지 (중소벤처기업부)

예비 창업자 대상, 최대 1억 원 지원.

| 심사 항목 | 배점 | 세부 평가 요소 |
|-----------|------|---------------|
| 문제 인식 | 15점 | 고객 문제 정의, 시장 기회 발견, 문제 해결의 시급성 |
| 실현 가능성 | 25점 | 기술/서비스 구현 능력, 프로토타입, MVP, 개발 계획 |
| 성장 전략 | 25점 | 시장 규모, 고객 확보 전략, 수익 모델, 확장 가능성 |
| 팀 구성 | 20점 | 대표자 역량, 팀원 전문성, 역할 분담 적정성 |
| 사회적 가치 | 15점 | 일자리 창출, 사회문제 해결, ESG, 지역 경제 기여 |

**예비창업패키지 전략 포인트:**
- 아직 창업 전이므로, "문제 인식"과 "실현 가능성"이 핵심이다
- 고객 인터뷰, 설문 등 시장 검증(Validation) 경험이 있으면 강력한 근거가 된다
- 팀 구성이 아직 미비해도, 향후 채용 계획을 구체적으로 기술한다
- 기술보다는 "고객 문제 해결"에 초점을 맞춘 서술이 효과적이다
- 린 스타트업(Lean Startup) 방법론 기반의 접근이 심사위원에게 친숙하다

### 4. 창업성장기술개발 (중소벤처기업부)

창업 7년 이내 중소기업 대상, 최대 연 2억 원 (2년).

| 심사 항목 | 배점 | 세부 평가 요소 |
|-----------|------|---------------|
| 기술 개발 능력 | 30점 | 기술 수준, 핵심 기술 보유 현황, 기술 인력 확보, 선행 연구 |
| 개발 계획 타당성 | 25점 | 연구 목표 명확성, 개발 방법론, 추진 일정 적정성, 산출물 구체성 |
| 사업화 계획 | 25점 | 시장 분석, 매출 추정 근거, 마케팅 전략, 판로 확보 방안 |
| 기술적 파급효과 | 10점 | 기술 확산 가능성, 관련 산업 파급, 표준화/인증 가능성 |
| 정책 기여도 | 10점 | 고용 창출, 수출 기여, 지역 발전, 사회적 가치 |

**창업성장기술개발 전략 포인트:**
- R&D 사업이므로 기술 개발 계획의 구체성과 체계성이 최우선이다
- 기존 보유 기술(TRL 수준)과 목표 기술 수준을 명확히 구분한다
- 개발 결과물의 성능 목표를 정량적으로 제시한다 (예: "정확도 95% 이상", "처리 속도 0.5초 이내")
- 사업비 예산 항목이 연구 내용과 정확히 대응해야 한다

### 5. 중소기업기술혁신개발 (중소벤처기업부/TIPA)

중소기업 대상 R&D 지원, 시장대응형/전략형 등 다수 트랙.

| 심사 항목 | 배점 | 세부 평가 요소 |
|-----------|------|---------------|
| 기술 개발의 필요성 | 15점 | 국내외 기술 동향, 기존 기술 한계, 개발 당위성 |
| 기술 개발의 우수성 | 30점 | 기술 독창성, 개발 목표 수준, 핵심 기술 확보 전략, IP 전략 |
| 개발 계획의 적정성 | 20점 | 추진 체계, 일정, 연구 인력, 장비, 사업비 적정성 |
| 사업화 가능성 | 25점 | 목표 시장, 매출 계획, 양산 계획, 인허가 계획 |
| 정책 부합성 | 10점 | 산업 정책 방향, 수출/고용 기여도, 기술 파급효과 |

**중소기업기술혁신개발 전략 포인트:**
- "기술 우수성" 배점이 가장 높으므로, 기술 차별화 논거에 집중한다
- 선행 기술 대비 비교표(성능, 가격, 크기 등)를 반드시 포함한다
- 트랙(시장대응형, 전략형)에 따라 강조점이 다르다:
  - 시장대응형: 단기 사업화(2년 이내 매출) 가능성 강조
  - 전략형: 기술 난이도와 장기 파급효과 강조
- 참여기관/위탁기관이 있는 경우, 역할 분담의 합리성을 기술한다

### 6. 초기창업패키지 (중소벤처기업부)

창업 3년 이내 기업 대상, 최대 1억 원 지원.

| 심사 항목 | 배점 | 세부 평가 요소 |
|-----------|------|---------------|
| 문제 인식 및 아이디어 | 20점 | 고객 Pain Point 정의, 솔루션의 혁신성, 시장 기회 |
| 기술/제품 실현 가능성 | 25점 | 프로토타입 완성도, 기술적 구현력, 개발 로드맵 |
| 시장성 및 성장성 | 25점 | TAM/SAM/SOM 분석, 경쟁 환경, BM 수익 구조 |
| 대표자/팀 역량 | 20점 | 관련 경력, 팀 보완성, 실행력 |
| 사회적 가치 | 10점 | 고용 창출, ESG, 사회문제 해결 기여도 |

### 7. 민관공동창업자발굴육성 (창업진흥원)

기술 사업화 대상, 정부+민간 매칭.

| 심사 항목 | 배점 | 세부 평가 요소 |
|-----------|------|---------------|
| 기술 사업화 가능성 | 30점 | 기술 성숙도(TRL), 시제품 수준, 기술 보호 전략 |
| 사업 모델 | 25점 | 수익 구조, 고객 유형, 가격 전략, 영업 채널 |
| 시장 분석 | 20점 | 목표 시장 크기, 경쟁 분석, 시장 진입 장벽 |
| 팀 역량 | 15점 | 핵심 인력, 자문단, 파트너십 |
| 투자 연계성 | 10점 | 투자 유치 가능성, 매칭 투자 확보 여부 |

## 심사기준 ↔ 양식 섹션 매핑 전략

```
심사기준 항목        양식 섹션                  어필 전략
───────────────    ──────────────────        ───────────────
기술성 (30점)    → 기술개발 내용 (섹션 3,4)   → 독창성, 선행연구, 특허
사업성 (25점)    → 사업화 전략 (섹션 5,6)     → 시장규모, 매출추정, 고객확보
대표역량 (20점)  → 기업현황 (섹션 2)          → 경력, 실적, 인력
타당성 (15점)    → 사업비, 추진일정 (섹션 7)  → 예산 합리성, 마일스톤
정책부합 (10점)  → 사업개요 (섹션 1)          → 정부 정책 키워드 반영
```

## 심사위원 관점

### 심사위원이 좋아하는 것
- 한눈에 파악되는 표와 차트
- 구체적인 수치와 출처
- 논리적인 흐름 (문제 → 해결 → 효과)
- 깔끔한 양식 준수
- 공고문 심사 기준에 정확히 대응하는 구성
- 기존 기술/제품 대비 정량적 비교표
- 명확한 마일스톤과 산출물 정의
- 팀 구성원의 역할이 기술 개발 내용과 매칭되는 것

### 심사위원이 싫어하는 것
- 뜬구름 잡는 추상적 서술
- 출처 없는 수치
- 빈칸이나 "추후 작성"
- 양식과 다른 임의 서식 변경
- AI가 쓴 것 같은 반복적이고 피상적인 표현
- 기술 내용과 무관한 사업비 항목
- 과도하게 낙관적인 매출 전망 (근거 없이 매년 200~300% 성장)
- 심사 기준과 무관한 내용으로 분량만 채운 서술
- 지나치게 많은 사업을 동시에 추진하는 비현실적 계획

## 발표 평가 대비

일부 사업(창업도약패키지, TIPS 등)은 서면 평가 후 발표 평가를 진행한다.

### 발표 평가 핵심 원칙

| 항목 | 가이드 |
|------|--------|
| 발표 시간 | 보통 7~10분 (초과 시 감점) |
| Q&A | 5~10분 (질문 의도를 정확히 파악하고 간결히 답변) |
| 자료 형식 | PPT, 보통 15~20슬라이드 |
| 핵심 구성 | 문제 정의 → 솔루션 → 시장 → 팀 → 재무 계획 |

### 발표 시 주의사항

1. 사업계획서와 발표자료의 수치가 반드시 일치해야 한다
2. 슬라이드당 핵심 메시지 1개, 텍스트는 최소화하고 시각 자료 활용
3. 데모(시연)가 가능하면 적극 활용한다
4. 질의응답에서 모르는 것은 솔직히 인정하되, 대안을 제시한다
5. 예상 질문 리스트(최소 20개)를 준비하고 답변을 연습한다

## 배점별 작성 분량 가이드

| 배점 | 권장 분량 (A4 기준) | 시각 자료 | 서술 깊이 |
|------|-------------------|----------|----------|
| 30점 이상 | 3~5페이지 | 표 2~3개, 차트 1~2개 | 심층 분석, 데이터 기반 논거 |
| 20~29점 | 2~3페이지 | 표 1~2개, 차트 1개 | 구체적 계획, 근거 제시 |
| 10~19점 | 1~2페이지 | 표 1개 | 핵심 요약, 간결한 서술 |
| 10점 미만 | 0.5~1페이지 | 필요 시 표 1개 | 요점만 간략히 |

## 가점 항목 상세

주요 사업에서 공통적으로 적용되는 가점 항목을 정리한다.

### 공통 가점 항목

| 가점 항목 | 배점 범위 | 증빙 서류 | 비고 |
|-----------|----------|----------|------|
| 지식재산권 (특허, 실용신안) | 1~3점 | 특허 등록증 또는 출원 확인서 | 등록 > 출원, 관련 기술에 한함 |
| 인증 보유 (ISO, CE, KC 등) | 1~2점 | 인증서 사본 | 유효 기간 내 인증에 한함 |
| 고용 계획/실적 | 1~3점 | 고용보험 가입자 명부 | 신규 채용 계획 포함 |
| 여성 기업 | 1~2점 | 여성기업 확인서 | 여성기업지원법 기준 |
| 장애인 기업 | 1~2점 | 장애인기업 확인서 | |
| 사회적 기업 | 1~2점 | 사회적기업 인증서 | |
| 지방 소재 기업 | 1~2점 | 사업자등록증(지방 주소) | 수도권 외 소재 기업 |
| 기술 보증 | 1점 | 기술보증기금 보증서 | |
| 벤처 인증 | 1~2점 | 벤처기업 확인서 | 유효 기간 내 |
| 수출 실적 | 1~2점 | 수출 실적 증명원 | 직수출 기준 |

### 가점 활용 전략

1. **가점은 합격/불합격의 경계에서 결정적 역할**을 하므로, 해당하는 모든 가점을 챙겨야 한다
2. 가점 증빙은 신청 마감일 기준으로 유효해야 한다
3. 특허 출원은 비용과 시간이 필요하므로, 사업 신청 전 미리 출원하는 것이 전략적이다
4. 고용 가점은 실제 채용 예정 인원을 과대 기재하면 추후 이행 점검에서 불이익이 있다

## 과락 기준 주의사항

| 사업명 | 과락 기준 | 비고 |
|--------|----------|------|
| 중소기업기술혁신개발 | 각 대항목 60% 미만 시 탈락 | 기술성 30점 중 18점 미만이면 탈락 |
| 창업도약패키지 | 총점 60점 미만 탈락 | 대항목별 과락은 없으나 총점 기준 |
| TIPS | 운영사별 자체 기준 | 기술성 과락 기준이 높은 편 |
| 창업성장기술개발 | 총점 60점 미만 및 대항목 과락 | 사업별 상이 |

**과락 방지 원칙**: 어떤 항목도 "0점"이 되지 않도록, 모든 평가 항목에 최소한의 내용을 반드시 작성한다. 특정 항목에 해당 없는 경우라도 관련 계획이나 방향성을 기술하여 배점을 확보한다.
