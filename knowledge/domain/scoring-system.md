# 심사 배점 체계

## 개요

정부 지원사업 사업계획서 심사는 **정량 평가**와 **정성 평가**로 구성된다.
배점표를 정확히 분석하고, 고배점 항목에 전략적으로 집중하는 것이 핵심이다.

---

## 일반적인 심사 항목 구조

대부분의 정부 지원사업은 다음과 같은 대항목으로 구성된다:

### 공통 심사 항목

| 대항목 | 일반 배점 범위 | 핵심 포인트 |
|--------|-------------|------------|
| 기술성 | 20~35점 | 기술의 차별성, 개발 가능성, 독창성 |
| 사업성 | 15~30점 | 시장 규모, 매출 계획, 사업화 전략 |
| 대표자/팀 역량 | 10~25점 | 경력, 실적, 전문성, 팀 구성 |
| 사업 타당성 | 10~20점 | 예산 적정성, 추진 일정, 실현 가능성 |
| 정책 부합성 | 5~15점 | 정부 정책 방향과의 일치도 |
| 가점 | 0~10점 | 특허, 인증, 고용, 지역, 여성/장애인 등 |

### 배점 분석 원칙

1. **고배점 항목 우선** — 30점짜리 기술성에 40%의 노력 투자
2. **감점 방지** — 빈칸, 오탈자, 수치 불일치는 직접 감점
3. **가점 극대화** — 해당되는 가점 항목 빠짐없이 챙기기
4. **과락 기준 확인** — 특정 항목 60% 미만이면 탈락하는 경우 있음

## 심사기준 ↔ 양식 섹션 매핑 전략

```
심사기준 항목        양식 섹션                  어필 전략
───────────────    ──────────────────        ───────────────
기술성 (30점)    → 기술개발 내용 (섹션 3,4)   → 독창성, 선행연구, 특허
사업성 (25점)    → 사업화 전략 (섹션 5,6)     → 시장규모, 매출추정, 고객확보
대표역량 (20점)  → 기업현황 (섹션 2)          → 경력, 실적, 인력
타당성 (15점)    → 사업비, 추진일정 (섹션 7)  → 예산 합리성, 마일스톤
정책부합 (10점)  → 사업개요 (섹션 1)          → 정부 정책 키워드 반영
```

## 심사위원 관점

### 심사위원이 좋아하는 것
- 한눈에 파악되는 표와 차트
- 구체적인 수치와 출처
- 논리적인 흐름 (문제 → 해결 → 효과)
- 깔끔한 양식 준수

### 심사위원이 싫어하는 것
- 뜬구름 잡는 추상적 서술
- 출처 없는 수치
- 빈칸이나 "추후 작성"
- 양식과 다른 임의 서식 변경
- AI가 쓴 것 같은 반복적이고 피상적인 표현

## 배점별 작성 분량 가이드

| 배점 | 권장 분량 (A4 기준) | 시각 자료 |
|------|-------------------|----------|
| 30점 이상 | 3~5페이지 | 표 2~3개, 차트 1~2개 |
| 20~29점 | 2~3페이지 | 표 1~2개, 차트 1개 |
| 10~19점 | 1~2페이지 | 표 1개 |
| 10점 미만 | 0.5~1페이지 | 필요 시 표 1개 |
