# 정부 사업계획서 문체 가이드

## 기본 원칙

정부 지원사업 사업계획서는 **심사위원이 읽는 공식 문서**다.
학술 논문도 아니고, 마케팅 자료도 아니다. 그 중간의 '관(官) 문체'가 필요하다.

---

## 문체 규칙

### 1. 문장 종결

- **~한다/~이다/~있다** 체 사용 (합니다체 금지)
- 예: "본 사업을 통해 생산성을 30% 향상시킨다" (O)
- 예: "본 사업을 통해 생산성을 30% 향상시킵니다" (X)

### 2. 주어

- **"본 사업"**, **"당사"**, **"본 과제"** 등 공식 주어 사용
- "저희 회사", "우리" 등 구어체 금지
- 주어 생략 시 문맥상 명확해야 함

### 3. 수치 표현

- 구체적 수치 + 근거 필수: "국내 시장 규모는 약 2조 5,000억 원이다(출처: 통계청, 2025)"
- 모호한 표현 금지: "많은", "상당한", "크게 증가" → 구체적 수치로 대체
- 단위 통일: 한 문서 내에서 천원/백만원/억원 중 하나로 통일
- 소수점: 백분율은 소수점 1자리, 금액은 정수

### 4. 문단 구성

- 한 문단 = 한 핵심 메시지
- 두괄식 (핵심 → 설명 → 근거)
- 문단 길이: 3~5문장 적정
- 불릿 포인트와 서술형 적절히 혼합

### 5. 피해야 할 표현

| 피할 것 | 대체 표현 |
|---------|----------|
| 최고의, 최첨단의 | 높은 수준의, 선도적인 |
| 혁신적인 (남발) | 차별화된, 개선된 |
| ~할 것입니다 | ~할 계획이다, ~을 추진한다 |
| 매우, 아주, 굉장히 | (삭제하거나 수치로 대체) |
| 기존에 없던 | 기존 대비 차별화된 |

### 6. 강조 기법

- **볼드체**: 핵심 키워드, 핵심 수치에만 사용
- 밑줄: 사용 자제 (가독성 저하)
- 색상: 표의 헤더나 강조 셀에만 제한적 사용

## 섹션별 문체 특징

### 사업 개요
- 간결하고 명확하게, 사업의 필요성과 목적을 1~2문단으로
- "~의 필요성이 대두되고 있다" → "~가 시급하다/필요하다"

### 기술 개발 내용
- 기술적 구체성 + 도식/표 병행
- 전문 용어는 괄호 안 설명 추가
- "본 기술은 A 방식을 적용하여 B 문제를 해결한다"

### 사업화 전략
- 시장 데이터 기반 논거
- TAM → SAM → SOM 순서로 시장 규모 제시
- 매출 추정은 보수적 시나리오 포함

### 대표자 역량
- 팩트 기반 (경력, 특허, 수상, 인증)
- 자기 자랑 아닌 객관적 실적 나열

## 분량 가이드

- 글자 수 제한이 있으면 90~95% 채우기 (100% 넘기면 잘림)
- 페이지 제한이 있으면 마지막 페이지 70% 이상 채우기
- 빈 칸 없이 모든 항목 작성 (해당 없음 시 "해당 없음" 명시)
